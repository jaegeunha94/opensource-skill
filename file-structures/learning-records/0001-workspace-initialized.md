# 파일구조 학습 워크스페이스 초기화

파일구조(File Structures) 과목 학습을 Day 1부터 시작함. 학습자는 한국 대학 CS 커리큘럼 기준으로 파일 조직 원리, 인덱싱, 해싱, B-트리를 체계적으로 익히는 것이 목표다.

## Evidence

Day 1 레슨(파일구조 개요 및 기본 개념)을 생성하여 학습 시작. 필드·레코드·파일·DB 계층 구조, 주기억장치 vs 보조기억장치 차이, 세 가지 파일 조직 방법 개요를 다룸.

## Implications

Day 2부터는 세 가지 파일 조직 방법을 순서대로 심화 학습한다: 순차 파일 → 인덱스 구조 → 직접 파일(해싱) → B-트리/B+-트리 순.
