# 파일구조 Resources

## Knowledge

- [Book: _File Structures: An Object-Oriented Approach with C++_ — Folk, Zoellick & Riccardi](https://www.pearson.com/en-us/subject-catalog/p/file-structures-an-object-oriented-approach-with-c/P200000003416)
  파일구조 과목의 대표 교재. 레코드, 순차 파일, 인덱싱, 해싱, B-트리 전반을 다룸. 핵심 개념 학습에 사용.

- [Book: _Database System Concepts_ — Silberschatz, Korth & Sudarshan (7판)](https://db-book.com/)
  데이터베이스 교재이나 파일 조직, 인덱싱, B+-트리 챕터가 파일구조와 직결됨. 인덱스 및 트리 구조 심화 시 참고.

- [강의: 한국방송통신대학교 파일처리론 — KNOU OpenCourseWare](https://ucampus.knou.ac.kr/)
  국내 파일구조/파일처리 과목의 표준 커리큘럼. 순차·인덱스·직접 파일, B-트리 순서로 구성됨.

- [Lecture Notes: CS 143 — Stanford Database Systems](https://cs.stanford.edu/people/widom/cs145/)
  스탠퍼드 데이터베이스 과목 자료. 파일 조직과 인덱스 섹션 참고.

- [Article: "B-Tree Visualization" — USFCA Data Structure Visualizations](https://www.cs.usfca.edu/~galles/visualization/BTree.html)
  B-트리 삽입/삭제를 단계별로 시각화. B-트리 Day 학습 시 필수 도구.

- [Article: "How does a relational database work" — coding-geek.com](http://coding-geek.com/how-databases-work/)
  실제 데이터베이스 내부에서 파일구조가 어떻게 쓰이는지 설명. 학습 동기 부여에 유용.
