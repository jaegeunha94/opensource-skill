HTTP 상태 코드 — 실무 판단 기준

코드의미쓰는 상황
200OKGET 성공, PUT/PATCH 성공
201CreatedPOST로 리소스 생성 성공
204No ContentDELETE 성공 (바디 없음)
400Bad Request입력 검증 실패 (클라이언트 잘못)
401Unauthorized인증 없음 또는 토큰 만료
403Forbidden인증은 됐지만 권한 없음
404Not Found리소스가 존재하지 않음
409Conflict중복 생성, 낙관적 잠금 충돌
422UnprocessableFastAPI 기본 검증 오류 (Pydantic)
429Too Many RequestsRate limit 초과
500Internal Server Error예상치 못한 서버 오류
503Service UnavailableDB 다운, 의존 서비스 불가
