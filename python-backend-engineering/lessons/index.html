Backend 구성을 위한 Python 실무자 기준 학습

Python 기본 문법이 아닌, 실무 백엔드 시스템을 설계하고 운영하는 역량을 다룹니다.

  • Day 1
    Python 백엔드 아키텍처 전체 구조 개요
    요청 수명 주기, 프레임워크 선택 기준, 실무 프로젝트 구조
  • Day 2
    API 설계 원칙 — REST, OpenAPI, FastAPI 실전
    HTTP 메서드 의미론, 상태 코드, Pydantic v2 스키마, 자동 문서화
  • Day 3
    데이터 모델링과 ORM — SQLAlchemy 2.x, Alembic 실전
    Data Mapper 패턴, 관계 매핑, N+1 문제와 로딩 전략, 무중단 마이그레이션
  • Day 4
    서비스 경계 설계 — 레이어드 아키텍처, Repository 패턴
    레이어드 아키텍처, Repository 패턴, 도메인 서비스 vs 응용 서비스, DI로 레이어 조립
  • Day 5
    트랜잭션과 동시성 — ACID, 격리 수준, 잠금 전략
    ACID, 격리 수준, 낙관적/비관적 잠금, 데드락 방지
  • Day 6
    인증과 인가
    JWT, OAuth2, RBAC
  • Day 7
    백그라운드 작업
    Celery, Redis Queue, 재시도 전략
  • Day 8
    관측성
    구조화 로깅, Prometheus, OpenTelemetry
  • Day 9
    배포와 운영
    Docker, Compose, CI/CD, 헬스체크
