# 자료구조 학습 시작 — 기준선 확립

자료구조 과목 학습을 Day 1부터 시작. ADT 개념, Big-O 표기법, 자료구조 전체 맵을 첫 레슨으로 다루었다. 이후 모든 레슨은 "어떤 자료구조를 선택할 것인가"라는 질문에 답할 수 있도록 시간/공간 복잡도를 비교하는 방향으로 진행한다.

## Implications

- Day 2부터 배열부터 시작해 선형 자료구조를 순서대로 다룬다
- 각 자료구조 레슨은 복잡도 분석을 필수로 포함한다
- 코딩 테스트 대비를 미션으로 하므로, 레슨마다 의사코드 또는 Python 예제를 반드시 포함한다
