# 자료구조 Resources

## Knowledge

- [Book: _Introduction to Algorithms_ (CLRS) — Cormen, Leiserson, Rivest, Stein](https://mitpress.mit.edu/9780262046305/)
  알고리즘·자료구조의 표준 교재. 각 자료구조의 수학적 분석과 의사코드를 담고 있다. 시간복잡도 증명이 필요할 때 참조.

- [Book: _Data Structures and Algorithm Analysis in Java_ — Mark Allen Weiss](https://users.cs.fiu.edu/~weiss/)
  C/Java 구현 중심. 트리·해시·정렬·그래프를 실용적으로 다룬다. 구현 참고용.

- [Course: MIT 6.006 Introduction to Algorithms (OpenCourseWare)](https://ocw.mit.edu/courses/6-006-introduction-to-algorithms-fall-2011/)
  MIT 공개강의. 강의 노트와 문제 세트 포함. 영상으로 개념을 빠르게 훑을 때 사용.

- [Course: CS50x — Harvard (edX)](https://cs50.harvard.edu/x/)
  C 언어 기반으로 연결 리스트·트리·해시 테이블을 직접 구현. 완전 초보자에게도 접근하기 쉬움.

- [Site: Visualgo — Data Structure Visualizations](https://visualgo.net/ko)
  자료구조 동작을 애니메이션으로 시각화. 정렬·트리·그래프·힙 모두 지원. 개념 이해 단계에서 필수.

- [Site: Big-O Cheat Sheet](https://www.bigocheatsheet.com/)
  자료구조별 시간·공간 복잡도 표. 레퍼런스로 즐겨찾기에 추가 권장.

- [Book: _Grokking Algorithms_ — Aditya Bhargava](https://www.manning.com/books/grokking-algorithms)
  그림 중심 입문서. 복잡도 직관, 탐색, 그래프까지 가볍게 읽을 수 있음. Day 1–5 병행 독서 추천.

## Wisdom (Communities)

- [r/learnprogramming](https://reddit.com/r/learnprogramming)
  자료구조 학습 중 막히는 부분에 대한 질문 스레드가 많다. 검색 후 질문하면 빠른 답변 가능.

- [Stack Overflow — [data-structures] tag](https://stackoverflow.com/questions/tagged/data-structures)
  구현 관련 구체적인 질문에 최적. "왜 이 코드가 느린가"류 질문에 상세 답변 다수.

- [LeetCode Discuss](https://leetcode.com/discuss/)
  코딩 테스트 문제별 풀이 토론. 자료구조 선택 이유를 다양한 관점에서 볼 수 있음.
