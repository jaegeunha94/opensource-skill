# 자료구조 Resources

## Knowledge

- [Book: _Introduction to Algorithms_ (CLRS) — Cormen, Leiserson, Rivest, Stein](https://mitpress.mit.edu/9780262046305/introduction-to-algorithms/)
  자료구조와 알고리즘의 표준 교재. 엄밀한 수학적 정의와 증명. Use for: 복잡도 분석, 각 자료구조의 정확한 정의, 고급 개념 검증.

- [Book: _Data Structures and Algorithm Analysis in C++_ — Mark Allen Weiss](https://www.pearson.com/en-us/subject-catalog/p/data-structures-and-algorithm-analysis-in-c/P200000003452)
  학부 수준의 자료구조 교재. 구현 중심. Use for: 구체적 구현 패턴, 연습 문제.

- [MIT OpenCourseWare: 6.006 Introduction to Algorithms](https://ocw.mit.edu/courses/6-006-introduction-to-algorithms-spring-2020/)
  MIT 알고리즘 강의 (자료구조 포함). 무료 강의 영상과 문제집 제공. Use for: 각 자료구조의 이론적 배경, 복잡도 증명.

- [Visualgo — 자료구조 시각화](https://visualgo.net/en)
  자료구조 동작을 애니메이션으로 시각화. 삽입/삭제/탐색 과정을 단계별로 확인. Use for: 직관적 이해, 알고리즘 동작 확인.

- [CS50x — Week 5: Data Structures (Harvard)](https://cs50.harvard.edu/x/)
  하버드 CS50 5주차. 배열, 연결 리스트, 해시 테이블, 트리를 C 코드로 직접 구현. Use for: 저수준 구현 이해, 포인터 기반 자료구조.

- [Python Documentation: collections module](https://docs.python.org/3/library/collections.html)
  Python 표준 라이브러리의 자료구조 구현. Use for: 실용적 사용 패턴, deque, OrderedDict, Counter 활용.

## Wisdom (Communities)

- [r/learnprogramming](https://www.reddit.com/r/learnprogramming/)
  학습 관련 질문과 토론. 자료구조 개념 이해 질문에 좋은 답변을 많이 볼 수 있음. Use for: 개념 확인, 학습 방법 조언.

- [LeetCode Discuss](https://leetcode.com/discuss/)
  코딩 문제 풀이 토론. 각 문제에서 어떤 자료구조를 선택하고 이유가 무엇인지 다양한 접근법 확인. Use for: 실전 문제 적용 패턴, 면접 준비.

- [Stack Overflow — data-structures tag](https://stackoverflow.com/questions/tagged/data-structures)
  구현 관련 구체적 질문과 답변. Use for: 특정 구현 문제 해결.
