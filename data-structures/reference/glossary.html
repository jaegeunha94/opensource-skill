자료구조 용어집

Reference 빠른 참조 — 학습하면서 계속 추가됨

A

ADT (Abstract Data Type, 추상 자료형)
구현 세부사항을 숨기고, 허용된 연산의 인터페이스만 정의한 개념적 모델. 예: 스택 ADT는 push/pop/peek을 정의하며, 배열로 구현할지 연결 리스트로 구현할지는 명시하지 않는다.

B

Big O Notation (빅오 표기법)
입력 크기 n에 따른 알고리즘 시간·공간 복잡도의 점근적 상한을 나타내는 표기법. 상수 계수와 하위 항을 무시하고 지배적인 항만 표기한다. O(1), O(log n), O(n), O(n log n), O(n²) 등.

D

Data Structure (자료구조)
데이터를 효율적으로 저장·탐색·수정·삭제하기 위한 체계적 방식. 선형(배열, 리스트, 스택, 큐)과 비선형(트리, 그래프)으로 나뉜다.

L

Linear Data Structure (선형 자료구조)
데이터 요소가 순서대로 한 줄로 나열된 자료구조. 배열, 연결 리스트, 스택, 큐가 해당된다.

N

Non-Linear Data Structure (비선형 자료구조)
데이터 요소가 계층(트리) 또는 그물(그래프) 구조로 연결된 자료구조. 선형 자료구조와 달리 하나의 요소가 여러 요소와 연결될 수 있다.
