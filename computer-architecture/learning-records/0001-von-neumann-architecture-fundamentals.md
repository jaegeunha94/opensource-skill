# 폰 노이만 아키텍처 핵심 개념 학습 (Day 1)

Day 1 레슨에서 컴퓨터 구조의 세 층위(Architecture / Micro-architecture / Logic Design),
폰 노이만 아키텍처의 핵심 구성 요소(ALU, 제어장치, 레지스터, 메모리, I/O),
그리고 Fetch–Decode–Execute 명령어 사이클과 폰 노이만 병목을 학습했습니다.

## Implications

- Day 2에서는 CPU가 숫자를 표현하는 방법(2진수, 보수법, 부동소수점)으로 넘어갈 수 있음
- 폰 노이만 병목 개념을 이해했으므로 캐시 메모리(Day 6~8 예정) 학습 시 동기 부여가 높을 것
- 레지스터 vs 메모리 속도 차이가 나중에 컴파일러 최적화(변수 할당) 이해에 연결됨
