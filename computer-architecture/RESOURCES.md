# Computer Architecture Resources

## Knowledge

- [Book: _Computer Organization and Design_ — Patterson & Hennessy (RISC-V Edition)](https://www.elsevier.com/books/computer-organization-and-design-risc-v-edition/patterson/978-0-12-820331-6)
  컴퓨터 구조의 표준 교과서. RISC-V ISA 기반. CPU 설계, 파이프라이닝, 메모리 계층 전반에 사용.

- [Book: _Computer Architecture: A Quantitative Approach_ — Hennessy & Patterson](https://www.elsevier.com/books/computer-architecture/hennessy/978-0-12-811905-1)
  고급 교재. 성능 분석, 명령어 수준 병렬성, 메모리 계층 최적화에 사용.

- [Course: MIT 6.004 Computation Structures (OpenCourseWare)](https://ocw.mit.edu/courses/6-004-computation-structures-spring-2017/)
  디지털 논리부터 프로세서 설계까지. 무료 강의 자료와 실습 포함.

- [Course: CMU 15-213 / 18-213 Introduction to Computer Systems](https://www.cs.cmu.edu/~213/)
  시스템 프로그래머 관점의 컴퓨터 구조. CS:APP 교재 사용.

- [Book: _CS:APP (Computer Systems: A Programmer's Perspective)_ — Bryant & O'Hallaron](https://csapp.cs.cmu.edu/)
  프로그래머가 알아야 할 컴퓨터 시스템 지식. 캐시, 메모리, 파이프라이닝 실용적 관점 제공.

- [YouTube: Computerphile](https://www.youtube.com/user/Computerphile)
  복잡한 CS 개념을 시각적으로 설명. 컴퓨터 구조 관련 영상 다수 포함.

- [Website: nand2tetris](https://www.nand2tetris.org/)
  NAND 게이트부터 컴퓨터 전체를 직접 만들어보는 실습형 강의. 디지털 논리 → 컴퓨터 구조 흐름 이해에 탁월.

## Wisdom (Communities)

- [r/computerarchitecture](https://www.reddit.com/r/computerarchitecture/)
  컴퓨터 구조 전문 서브레딧. 학술적 논의와 질문 가능.

- [Stack Overflow - computer-architecture tag](https://stackoverflow.com/questions/tagged/computer-architecture)
  구체적인 기술 질문 답변. 면접 준비 및 개념 확인에 유용.
