# UI/UX 용어 Resources

## Knowledge

- [Nielsen Norman Group — UX Articles & Reports](https://www.nngroup.com/articles/)
  세계 최고 UX 리서치 기관의 공식 아티클. 용어 정의, 원칙 설명 모두 신뢰도 최상급. 특정 용어의 정확한 의미를 확인할 때 1차 출처로 사용.

- [Interaction Design Foundation — UX Design Glossary](https://www.interaction-design.org/literature/topics/ux-design)
  용어별 개념 아티클과 커리큘럼이 체계적으로 정리되어 있음. 학습 경로 설계에 참고.

- [Material Design Guidelines — Google](https://m3.material.io/)
  구글의 공식 디자인 시스템 문서. 컴포넌트·패턴·토큰 등 실무 용어의 공식 정의가 담겨 있음.

- [Apple Human Interface Guidelines](https://developer.apple.com/design/human-interface-guidelines/)
  iOS/macOS 플랫폼 특화 용어와 패턴. 모바일 UI 용어를 다룰 때 참고.

- [Figma — Design Glossary](https://www.figma.com/resource-library/design-glossary/)
  피그마 팀이 관리하는 용어집. 협업 툴 맥락에서 쓰는 표현 기준으로 사용.

## Wisdom (Communities)

- [r/userexperience](https://www.reddit.com/r/userexperience/)
  UX 실무자들이 활발히 토론하는 서브레딧. 용어 혼용 사례나 현장 해석 차이를 확인할 때 유용.

- [Designer Hangout (Slack)](https://www.designerhangout.co/)
  초청제 UX 전문가 커뮤니티. 심화 용어 논쟁과 트렌드를 접하기에 좋음.

## Gaps

- 한국어 UI/UX 용어의 공식 표준 문서 없음 — 국내 표준을 다룰 때는 네이버·카카오 디자인 가이드 아티클을 보조로 활용할 것
