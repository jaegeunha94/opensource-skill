# 학습 참고 자료

## 공식 문서

- [Redis 공식 문서](https://redis.io/docs/) — 명령어 레퍼런스, 자료구조, 운영 설정
- [Redis University](https://university.redis.com/) — Redis 공식 무료 코스 (RU101, RU102 등)
- [Redis Best Practices](https://redis.io/docs/manual/patterns/) — 공식 패턴 가이드

## 도서

- **Designing Data-Intensive Applications** (Martin Kleppmann) — 캐시, 복제, 일관성 모델을 시스템 관점에서 다루는 필독서
- **Redis in Action** (Josiah Carlson) — Redis 자료구조별 실무 패턴을 상세히 다룸
- **High Performance MySQL** (O'Reilly) — DB와 캐시 계층 설계를 함께 이해하기 위한 참고
