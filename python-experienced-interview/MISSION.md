# 파이썬 경력자 Interview 학습 미션

## 목표

Python 경력자 면접에서 평가받는 **언어 이해도**와 **실무 판단력**을 기른다.  
기본 문법은 이미 알고 있다고 가정하며, 인터뷰어가 기대하는 사고 구조와 답변 품질을 훈련한다.

## 대상 역량

면접관은 경력자에게 다음을 평가한다:

1. **Python 언어 내부를 아는가** — data model, object model, 실행 모델, 메모리 관리, GIL
2. **동시성 설계를 이해하는가** — threading, multiprocessing, asyncio, GIL 우회
3. **고급 패턴을 실무에 적용하는가** — iterator/generator, context manager, descriptor, decorator
4. **타입 시스템과 패키징을 다루는가** — typing, mypy, packaging, dependency management
5. **품질을 유지하는가** — testing, debugging, profiling, 유지보수성, 코드 리뷰 사고
6. **trade-off를 설명하는가** — 왜 그 선택을 했는지, 다른 방법의 단점은 무엇인지

## 전제 조건

- 기본 Python 문법 및 OOP 개념 숙지
- 실무 Python 개발 경험 (최소 2년 이상 수준)

## 학습 방식

각 레슨은 **면접 질문 → 답변 구조 → 상세 설명 → 모범 답변 → 약한 답변 → follow-up → rubric** 순서로 진행한다.  
어려운 개념은 전제 개념부터 쉬운 한국어로 먼저 설명한 뒤 인터뷰 언어로 연결한다.

## 완료 기준

- 각 레슨의 self-check rubric에서 "상" 수준을 달성
- 상위 follow-up 질문에 즉석 답변 가능
- 강한 답변 / 약한 답변의 차이를 스스로 설명 가능
