# 네트워크 학습 미션

## 목표

컴퓨터 네트워크의 핵심 개념과 프로토콜을 이해하고, 실제 네트워크 환경에서 발생하는 문제를 진단하고 해결할 수 있는 실력을 갖춘다.

## 구체적 목표

1. **OSI 7계층 / TCP/IP 모델** 구조를 완전히 이해하고 각 계층의 역할을 설명할 수 있다.
2. **핵심 프로토콜** (HTTP/HTTPS, TCP, UDP, IP, DNS, DHCP, ARP 등)의 동작 원리를 이해한다.
3. **라우팅과 스위칭** 개념을 이해하고 서브넷 계산을 직접 수행할 수 있다.
4. **보안 기초** (TLS/SSL, 방화벽, NAT 등)를 이해하고 안전한 통신 원리를 설명할 수 있다.
5. **네트워크 도구** (Wireshark, ping, traceroute, netstat 등)를 활용해 실제 트래픽을 분석할 수 있다.

## 학습 방향

- 이론과 실습을 병행하여 개념을 체득한다.
- Wireshark 패킷 캡처로 실제 프로토콜 동작을 직접 확인한다.
- 각 레슨 후 핵심 개념을 자신의 말로 설명할 수 있을 때까지 복습한다.
