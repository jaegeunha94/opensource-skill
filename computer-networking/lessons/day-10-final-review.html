연습 문제

문제 1 — 10.0.4.200/22 의 네트워크 주소와 호스트 수는?

풀이: /22 → 서브넷 마스크 255.255.252.0 (상위 22비트 = 1)
세 번째 옥텟 4 = 00000100 → 하위 2비트 마스킹 → 00000100 & 11111100 = 00000100 = 4
→ 네트워크 주소: 10.0.4.0, 브로드캐스트: 10.0.7.255
→ 호스트 수: 2^(32-22) − 2 = 1022개

문제 2 — 100개 호스트를 수용하는 최소 서브넷 프리픽스는?

풀이: 2^n − 2 ≥ 100 → n ≥ 7 → 호스트 비트 7개
→ 프리픽스 = 32 − 7 = /25 (최대 호스트 126개)

문제 3 — 172.16.0.0/12 에 속하는 IP는? (다음 중 하나 고르기)

보기: A) 172.31.255.255   B) 172.32.0.1   C) 173.0.0.1
정답: A
/12 → 앞 12비트 고정: 10101100.0000 = 172.0∼172.31.x.x
172.31.255.255 → 세 번째 옥텟 172.X에서 두 번째 옥텟 31 = 00011111 → 상위 4비트 0000 ✔
172.32.0.1 → 두 번째 옥텟 32 = 00100000 → 상위 4비트 0010 ✗

4. 핵심 공식 & 암기 포인트

주제공식 / 핵심 암기
서브넷 호스트 수 2^(32 − prefix) − 2 (네트워크 주소 + 브로드캐스트 제외)
서브넷 마스크 prefix 개수만큼 상위 비트를 1로 채움. /24 → 255.255.255.0
TCP Handshake 연결: SYN → SYN-ACK → ACK (3-way) / 종료: FIN → ACK → FIN → ACK (4-way)
DNS 조회 캐시 → 재귀 리졸버 → 루트 NS → TLD NS → 권한 NS
DHCP DORA Discover(브로드캐스트) → Offer → Request → Acknowledge
ARP IP를 알 때 MAC을 얻는 프로토콜. 브로드캐스트 요청 → 유니캐스트 응답
TLS 핸드셰이크 ClientHello → ServerHello + 인증서 → 키교환 → Finished (대칭키 교환 완료)
혼잡 제어 Slow Start(지수↑) → Congestion Avoidance(선형↑) → Fast Retransmit/Recovery
NAT 사설 IP + 포트 ↔ 공인 IP + 포트 변환. PAT(Port Address Translation)이 가장 일반적
Wi-Fi 세대 802.11b(11M) → g(54M) → n(600M) → ac(3.5G) → ax(9.6G, Wi-Fi 6)
